
### Mode d'utilisation

Git:
```sh
$ git clone https://github.com/bailobah/relevanc.git
```

Docker:
```sh
$ cd relevanc
$ docker-compose up --build
```

Vérifiez le déploiement en naviguant vers l'adresse :
http://localhost:4000/api/
